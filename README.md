# Stiller Record

Приложение для автоматической нарезки записей экрана по сделкам Binance Futures.

## Скачать

Последняя версия — во вкладке **[Releases](../../releases/latest)**: скачайте
`stiller_record.exe` и запускайте. Ничего устанавливать не нужно — всё (включая
ffmpeg) внутри одного файла.

## Авто-обновление

Приложение само проверяет GitHub при запуске: если вышла новая версия —
скачивает и обновляется автоматически.

## Использование

1. Запустите `stiller_record.exe`.
2. Средняя кнопка мыши — открыть настройки. Введите Binance API Key/Secret,
   выберите папку и время начала записи.
3. Перетащите видео в окно и нажмите «Нарезать».
